# -*- encoding: utf-8 -*-

数据源
    聚宽数据jqdatasdk, ...
数据表
    sqlite3
报警条件
    1) macd的diff和dea交叉
    2) ...
股票品种
    中证小盘500指数, 沪深300指数, 上证50指数, ...
k线周期
    1分钟, 5分钟, ...
报警时间
    依据"k线周期"的设置，启动报警程序。


软件功能
-----------------------------------
v1.0
    1) 数据下载:
        从jqdatasdk读取1m数据。
        程序运行期间，下载的数据放在sqlite3中。
    2) 计算macd交叉。
    3) 交叉后报警:
        播放音乐。
    4) 定时执行。
        周一 -- 周五
            09:30 -- 11:30
            13:00 -- 15:00
        其它时间，停止工作。

v1.1
    历史数据保存在本地:
        数据保存在sqlite3中。
        sqlite3中，1分钟的历史数据，每年一张数据表。
        每次程序启动时，从sqlite3中读取历史数据。
        之后，从jqdatasdk补充缺失的数据。
        程序退出前，当天下载的数据，写入sqlite3中。

v1.2
    k线的时间，设置为开始时间。
    聚宽数据，为结束时间。
        1m的2020-10-10T09:31 对应于 2020-10-10T09:30:00.000 --- 09:30:59.999
        5m的2020-10-10T09:35 对应于 2020-10-10T09:30:00.000 --- 09:34:59.999
    mt5、文华财经是开始时间。
        1m的2020-10-10T09:31 对应于 2020-10-10T09:31:00.000 --- 09:31:59.999
        5m的2020-10-10T09:35 对应于 2020-10-10T09:35:00.000 --- 09:39:59.999

v1.3
    5m数据，由1m数据聚合而成。
